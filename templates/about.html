About This Project

This system uses deep learning models to classify brain MRI images into four categories:

  • Glioma Tumor
  • Meningioma Tumor
  • Pituitary Tumor
  • No Tumor

The application allows users to upload MRI scans and analyze them using trained neural network models such as EfficientNet and ResNet. The system predicts the tumor type and assists in automated medical image analysis.
